Glossary
========

.. glossary::

    Dataset
        Set of audio signals and reference annotations associated to them.

    Container
        Class to store data and based on data type to be stored, provides meaningful and clear access to the data.

    Metadata
        Annotations for data

    Repository
        Container to store multiple data containers.

    Encoder
        Class to convert data from different type to the another.

    Processor
        Class to provide uniform API for processing data.

    ProcessingChain
        Chain of Processors, output of previous processor is fed to input of next processor in the chain.
